Help

  • This fun project is not meant to be a fully fledged task manager.
    It's is more suited focusing on a daily priory list or the next set of tasks (GTD).
    The randomization function is meant to overcome indecision inertia...
    i.e. If I can't decide what to do next, then perhaps the computer can push me in the right direction.
  • To add a new task: Enter task text into the input field - at page top - and press <ENTER>
    Your new task will be added, in the card, at the bottom of the list.
  • To mark a task as completed: Click anywhere on the task card.
    Marked tasks are dimmed and struck out
    Clicking the card again will mark the task incomplete.
  • To edit task text: Click the ⟲ icon in the card, then edit the text.
    Clicking outside the card, pressing tab or <ENTER> will apply the edit.
    NOTE: Editing a completed task will mark that task remove its completed status. NOTE: If the task text has not been changed, then <ENTER> will not exit 'edit' mode.
  • To delete a task: Click the × icon in the card.
    NOTE: Deleting a task will permanently remove a task from your list.
  • To Randomize list order: Click the ⥯ icon in the top controls.
    Incomplete tasks are shuffled randomly, while completed tasks moved to the bottom of the list.
  • To re-order tasks: <T.B.D.>
  • Tasks are backed up to browser local storage. Your task list will therefore persist between browser sessions. Deleting browser data, will delete your list.
